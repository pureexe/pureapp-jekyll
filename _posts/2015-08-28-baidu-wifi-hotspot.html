เรื่องราวที่ผมได้มาพบเจอกับโปรแกรมนี้ มันก็คือว่าด้วยการที่ผมย้ายเข้ามาอยู่หอแห่งใหม่ หอแห่งนี้ให้ใช้เพียง 1 รหัสต่อ 1 เครื่องเท่านั้น และ 1 คนจะได้เพียงรหัสเดียว หมายความว่า ผมไม่สามารถใช้ไวไฟกับคอมพิวเตอร์ไปพร้อมกับไวไฟในมือถือได้ เพื่อนของผมจึงได้แนะนำมาว่าให้ลองใช้โปรแกรม Baidu WiFi Hotspot ดู มันจะช่วยกระจายสัญญาณจากคอมพิวเตอร์ไปมือถือได้

ไม่รอช้า ผมได้โหลดโปรแกรมนี้มาติดตั้งคอมพิวเตอร์ของผม โดยไฟล์มีขนาดไม่ใหญ่นักเพียง 4.91 MB เท่านั้น  ใช้งานได้ทันทีไม่ต้องดาวน์โหลดข้อมูลเพิ่มเติม (เมื่อติดตั้งเสร็จแล้วประมาณ 12MB) ลงชื่อแสดงความเป็นเจ้าของโดย Baidu, Inc แต่แอบสะดุ้งชื่อโปรแกรมเล็กน้อย ขึ้นเป็น Baidu PC faster แทน 


เมื่อเปิดโปรแกรมขึ้นมา เครื่องจะเริ่มทำการปล่อย wifi ทันที สามารถนำมือถือเชื่อมต่อเข้ามาใช้งานได้เลย


หน้าตาที่ใช้เป็นภาษาอังกฤษแม้ว่าผมจะใช้ windows ภาษาไทยก็ตามแต่สามารถเปลี่ยนได้โดยการกดที่ปุ่มสามขีดด้านบน แล้วเลือกภาษาไทย



สามารถเปลี่ยน ชื่อ wifi หรือรหัสผ่านโดยการนำเม้าไปชี้ด้านหลังและคลิกรูปดินสอ


โดยถ้าเราคิดชื่อไม่ออก ทาง Baidu มีชื่อล่วงหน้าใช้เรา
โดยขอผมมีชื่อต่างๆดังนี้
404 network unavailable (404 เครือข่ายใช้ไม่ได้)
Feel like flying (รู้สึกเหมือนได้บิน)
My neighbor sucks! (เพื่อนบ้านเฮงซวย)
FBI surveillance van (รถสอดแนมของ FBI)
I'm so lonely (ฉันโสด)


โดยเมื่อเราใช้มือถือเชื่อมต่อเข้ามาจะเด้งอุปกรณ์เชื่อมต่อที่มุมจอ



เมื่อคลิกที่ตัวเลขที่แสดง จะแสดงข้อมูลของอุปกรณ์ที่เชื่อมต่อ


เมื่อกดรูปลูกศรชี้ลงข้างชื่อเครื่องสามารถ บล็อคไม่ให้ใช้ wifi ได้



โดยเมื่อนำเม้าชี้ชื่ออุปกรณ์จะมีข้อมูลของอุปกรณ์ขึ้นมาดังนี้


โปรแกรมนี้ยังสามารถดึงไฟล์จากมือถือได้ด้วย โดยเมื่อกด get file from phone แล้วมันจะบอกให้เข้า s.com


และนี่คือหน้าตา s.com เมื่อเข้าผ่านมือถือ


เมื่ออัปโหลดไฟล์ก็จะขึ้นที่มุมจอ โดยไฟล์จะอยู่ในโฟลเดอร์ BaiduWifi หน้าจอ


ภาพเมื่ออัปโหลดไฟล์จากมือถือ


ลองกด sent file to phone โดยโปรแกรมจะให้ลากหรือเลือกไฟล์


เมื่อลากไปแล้ว โปรแกรม บอกให้เข้าไปที่ g.com

ผมลองเข้าไปที่ g.com แล้วแต่ดันเจอหน้าหมุนติ้วค้าง รอประมาณ 5 นาทีแล้วไม่มีอะไรเปลี่ยนแปลงผมจึงข้ามส่วนนี้ไปแล้วกันนะครับ


โดยเราสามารถแชร์รหัสผ่านของ wifi เราผ่านเฟสบุ๊คได้ด้วย เมื่อกดเมนู facebook บริเวณปุม 3 เส้น


แต่ค่อนข้างหน้ากลัวไปนิดตรงที่ต้องล็อคอินถึงแชร์ได้


สรุปครับ
โปรแกรมนี้ใช้สะดวกมาก แก้ปัญหาของผมได้เลย ขณะใช้งานนี้พบเพียงปัญหาเดียวคือเข้า g.com แล้วค้าง คอมพิวเตอร์และมือถือผมยังปกติดีอยู่ มีแนวโน้มที่ผมจะแนะนำโปรแกรมนี้ใช้คนอื่นใช้ต่อ

ปล.ผมแค่แนะนำไม่ได้โฆษณานะครับ
