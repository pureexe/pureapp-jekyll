วันนี้ผมจะแนะนำให้ท่านรู้จักกับ Github Pages เป็นบริการดีๆ จาก Github โดยเพียงเราสร้าง repository ของเราให้มี branch ชื่อว่า gh-pages เราก็จะฝากเว็บฟรีได้แล้ว อีกทั้งผมยังมีเทคนิคเล็กๆน้อยๆ แนบไว้ในท้ายคลิปด้วย ลองติดตามดูกันครับ
