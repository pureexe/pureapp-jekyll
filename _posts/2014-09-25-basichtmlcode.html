หลังจากที่ผมได้เริ่มสอนเขียนแอป มีคนถามคำถาม จำนวนมากเข้ามา และผมได้เล็งเห็นว่า ผู้ที่ต้องการสร้างแอปส่วนใหญ่นั้นไม่เคยมีพื้นฐานด้านการเขียนโปรแกรมมาก่อน แต่ไม่เป็นไรครับผมจะสอนให้คุณใช้คำสั่งพื้นฐานเป็นจนกระทั่งสามารถสร้างแอปขั้นสูงเหมือนผมได้เอง

ก่อนอื่นเลยจากความเดิมตอนที่แล้วคงเห็นกันแล้วว่าผมนั้นเขียนแอปผ่านบล็อกโดยใช้ปุ่ม HTML


โดยสามารถเลือกปุ่ม HTML ได้ตอนที่กำลังเขียนจาก จุดนี้

ดังนั้นผมจะแนะนำภาษา HTML ให้รู้จักคร่าวๆก่อนนะครับ เพื่อให้ผู้อ่านสามารถประยุกต์ใช้โค้ด HTML ในการสร้างแอปได้ครับ

เอชทีเอ็มแอล (อังกฤษ: HTML: Hypertext Markup Language) เป็นภาษามาร์กอัปหลักในปัจจุบันที่ใช้ในการสร้างเว็บเพจ หรือข้อมูลอื่นที่เรียกดูผ่านทางเว็บเบราว์เซอร์ ซึ่งตัวโค้ดจะแสดงโครงสร้างของข้อมูล ในการแสดง หัวข้อ ลิงก์ ย่อหน้า รายการ รวมถึงการสร้างแบบฟอร์ม เชื่อมโยงภาพหรือวิดีโอด้วย โครงสร้างของโค้ดเอชทีเอ็มแอลจะอยู่ในลักษณะภายในวงเล็บสามเหลี่ยม




อ้างอิง Wikipedia
ภาษา HTML ใช้การเขียนในรูปแบบของแท็กดังนั้นเมื่อเปิดด้วยแท็กได้ต้องปิดด้วยแท็กนั้นเช่น <img> ต้องปิดด้วยแท็ก </img> (สังเกตุได้ว่าแท็กปิดจะมี / ด้านหน้า) ซึ่งต้องจับกันให้ครบคู่พอดี เพื่อการแสดงผลที่ถูกต้อง นอกจากนี้ภาษา HTML ยังมีสิ่งที่เรียกว่า แอตทริบิวต์(Attribute) ซึ่ง แอตทริบิวต์นี้ใช้บอกสิ่งที่แท็กควรจะทำ เช่น แท็ก <img> มักจะมีแอตทริบิวต์ชื่อ src เสมอ โดยวิธีเขียนแอตทริบิวต์คือ <ชื่อแท็ก ชื่อแอตทริบิวต์="ค่าแอตทริบิวต์"> และระหว่างแท็กยังสามารถใช้แสดงค่าได้อีกด้วย เช่น <a>hello</a> ก็จะแสดงคำว่า hello เป็นลิงค์

 ต่อไปรู้จักโค้ดพื้นฐานในภาษา HTML กันครับ
<img></img> เป็นโค้ดที่ใช้สำหรับแสดงรูปภาพ
โดยมีแอตทริบิวต์ที่ควรรู้คือ src ใช้สำหรับ อ้างอิงว่ารูปของเราอยู่ที่ url ไหน
วิธีใช้คือ <img src="url ของรูปภาพ"></img>
เช่นต้องการแสดงภาพนี้

สามารถใช้คำสั่ง html ได้ว่า <img src="https://upic.me/i/dq/newicon.jpg"></img> ถ้ารูป ยังไม่ได้ฝากไฟล์ต้องนำไปฝากไฟล์ด้วยนะครับ เพื่อให้ไฟล์เราออนไลน์อยู่ตลอดเวลาสามารถแสดงผลได้ทันทีเมื่อผู้ใช้เรียก
*คำเตือนการใช้คำสั่ง <img> ห้ามใช้คำสั่ง src ชี้ url ไปเว็บคนอื่นที่ไม่ได้เป็นเว็บฝากไฟล์เด็ดขาดการกระทำนี้ เรียกว่า Hotlink จะทำให้เซิฟเวอร์ของผู้ที่ถูก url ชี้ไปทำงานหนักขึ้น และท่านอาจโดนเล่นงานกลับด้วยการเปลี่ยนรูปบนเว็บคุณแบบเหตุการณ์เช่นนี้ก็เป็นได้

<a></a> เป็นโค้ดสำหรับใช้แสดงลิงค์
มีแอตทริบิวต์ที่ควรรู้ 2 อย่าง
href ใช้อ้างถึง url ของเว็บที่ให้ไป
target ใช้แสดงถึง รูปแบบการเด้งหน้าต่างเมื่อคลิก ในที่นี้ผมจะแนะนำแค่ target='_blank' คือเปิดเว็บโดยเปิดไปหน้าใหม่ ไม่ใช่การเปลี่ยนหน้าเดิม
วิธีใช้คือ <a href="url ของหน้าที่จะไป">ข้อความที่จะแสดง</a>
