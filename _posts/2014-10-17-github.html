--- layout: post title: Github ส่งเสริมให้เด็กๆเขียนโปรแกรม แจกของแพงให้ใช้ฟรี date: '2014-10-17T13:30:00.001+07:00' author: Pakkapon Phongthawee tags: - "ประกาศ" modified_time: '2015-09-27T19:22:51.567+07:00' blogger_id: tag:blogger.com,1999:blog-3103072889085475591.post-6094122191848883951 blogger_orig_url: http://www.pureapp.in.th/2014/10/github.html --- GitHub ร่วมกับเว็บพันธมิตรแจก ของให้เด็กๆ อย่างผมให้ใช้ของแพงฟรีเพื่อ สร้างสรรค์สุดยอดแอป
สมัครโดยใช้ อีเมล ของ โรงเรียนหรือมหาวิทยาลัยครับ โดยจะต้องเป็น .ac.th หรือ .edu

อ้างอิงจาก blognone

  • Atom.io ของฟรีจาก GitHub เอง
  • Bitnami บริการติดตั้งซอฟต์แวร์บนคลาวด์แบบคลิกเดียว ให้บริการแบบ Business 3 ฟรีหนึ่งปี (ปกติเดือนละ 49 ดอลลาร์)
  • CrowdFlower เข้าใช้บริการปรับคุณภาพข้อมูลโดยคน มีเครดิตให้ฟรี 50 ดอลลาร์
  • DigitalOcean ใช้ฟรี 100 ดอลลาร์
  • DNSSimple ได้แพ็กเกจ Bronze ฟรีสองปี ปกติเดือนละ 3 ดอลลาร์
  • GitHub ได้ใช้บริการแบบ Micro ตลอดช่วงเวลาที่ยังเป็นนักเรียน
  • HackHands บริการตอบคำถามเขียนโปรแกรม ใช้บริการได้ 25 ดอลลาร์
  • Namecheap ให้โดเมน .me และใบรับรอง SSL ฟรีหนึ่งปี
  • Orchestrate บริการฐานข้อมูลแบบ Rest API ให้บริการแบบนักพัฒนาฟรีตลอดช่วงเวลาที่เป็นนักเรียน ปกติ 49 ดอลลาร์ต่อเดือน
  • Screenhero บริการแชร์หน้าจอ ให้บริการฟรีตลอดช่วงเวลาที่เป็นนักเรียน ปกติ 9.99 ดอลลาร์ต่อเดือน
  • SendGrid ระบบอีเมล ให้บริการแพ็กเกจนักเรียน 4.95 ดอลลาร์ต่อเดือนฟรีหนึ่งปี
  • Stripe บริการรับจ่ายเงิน ไม่คิดค่าธรรมเนียม 1000 ดอลลาร์แรก
  • Travis Cl แพลตฟอร์มวางระบบ (continuous integration) ให้บริการแบบเป็นส่วนตัวฟรีตลอดช่วงเวลาที่เป็นนักเรียน ปกติเดือนละ 69 ดอลลาร์
  • Unreal Engine ใช้งานฟรีตลอดช่วงเวลาที่เป็นนักเรียน ปกติเดือนละ 19 ดอลลาร์

ขอให้พวกเพื่อนๆสนุกกับสิ่งของที่ด้ และสร้างสรรค์แอปสุดยอดขึ้นมาครับ
