อุปกรณ์ที่ต้องใช้ครับ
1. มือถือ (Android)
2. คอมพิวเตอร์
3. สาย USB
4. Chrome (ทั้งคอมพิวเตอร์และมือถือครับ)
5. Driver มือถือ (ถ้ายังไม่ได้ลง)

วิธีการครับ
ขั้นแรก เปิด ตั้งค่าของมือถือครับ เลื่อนไปล่างสุดเลือกเกี่ยวกับโทรศัพท์

แตะคำว่า หมายเลขบิวด์ รัวๆ จนขึ้นว่าคุณเป็นนักพัฒนาแล้ว ครับ

เข้าไปที่หน้าแรกของตั้งค่า จะเห็นว่ามีเมนู นักพัฒนาโผล่ขึ้นมาใหม่ให้เข้าไปครับ

จากนั้นให้เลือกเปิดแก้ไขข้อบกพร่อง USB ครับ

จากนั้นเสียบสาย USB ครับ

ต่อมาเปิด Chrome บนคอมครับ แล้วเข้า chrome://inspect จะเห็นว่าตอนนี้มีอุปกรณ์ต่ออยู่ครับ

จากนั้นเปิดเว็บที่เราจะทดสอบในChrome มือถือเลยครับ เช่นผมเปิดเว็บนี้

บนคอมก็จะเห็นว่าเราเปิดเว็บ pureapp.in.th ในมือถืออยู่ครับ

ลองกด inspect เพื่อเปิด console มา debug เว็บครับ

แค่นี้การทดสอบเว็บบนมือถือก็ไม่ใช่เรื่องยากเลยครับ สู้ๆนะครับ
