ถ่ายทอดสดหน้าจอของคุณไปยังหน้าเฟสบุ๊ค หน้ากลุ่ม หรือเพจของคุณ

แอปนี้เป็นแอปที่ช่วยให้ท่านสามารถถ่ายทอดสดหน้าจอคอมพิวเตอร์ของตัวเองขึ้นไปยังโปรไฟล์เฟสบุ๊ค เพจบนเฟสบุ๊คและกลุ่มบนเฟสบุ๊คได้ โดยเพียงแค่ท่านกดปุ่มเริ่มถ่ายทอดสดด้านล่างนี้ โดยท่านสามารถอ่านขั้นตอนการถ่ายทอดสดได้ที่ด้านล่าง

เริ่มถ่ายทอดสด

วิธีการถ่ายทอดสด

ท่านสามารถดาวน์โหลดโปรแกรม OBS เพื่อใช้ในการถ่ายทอดสดได้ที่ obsproject.com โดยจะเห็นด้านซ้ายมือมีปุ่มเขียนว่า Windows 7/8/10 เมื่อคลิกแล้วจะมีไฟล์ดาวน์โหลดลงมา คลิกเปิดและติดตั้งให้เรียบร้อย

จากนั้นทำการติดตั้งโปรแกรมให้คลิกที่ปุ่ม เริ่มถ่ายทอดสด ที่ด้านบนของเว็บแล้วจะพบกับกล่องดังภาพเด้งขึ้นมาให้ท่านเลือกว่าจะโพสการถ่ายทอดสดไปยังตัวเอง ไปยังเพื่อน ไปยังเพจ หรือไปยังกลุ่ม

หากในขั้นนี้ที่แถบ url จะขึ้นว่ามีป๊อบอัปที่ถูกบล็อคอยู่ ให้กดอนุญาตแสดงป๊อบอัป (always allow)แล้วกดปุ่ม เริ่มถ่ายทอดสด ใหม่อีกครั้ง

ในขั้นนี้จะมีหน้าต่างข้อมูลของการถ่ายทอดสดขึ้นมา ให้กรอกข้อมูลต่างๆเช่น ข้อความ ชื่อวิดีโอ แท็ก อารมณ์ ฯลฯ ให้เรียบร้อย แต่ขั้นนี้ยังไม่สามารถกดปุ่ม Go Live! เนื่องจากวิดีโอยังออฟไลน์อยู่

ให้เปิดโปรแกรม OBS ขึ้นมาจากนั้นเลือก การตั้งค่า > ตั้งค่าการออกอากาศ เลือก Live Stream แล้วเลือก Custom จากนั้นคัดลอก Server URL จากหน้าเว็บ วางในช่อง FMS URL และคัดลอกช่อง Stream Key มาวางในช่อง Stream Key

จากนั้นที่หน้าแรกของโปรแกรม OBS จะเห็นช่อง source เลือก Monitor capture (หรืออย่างอื่นก็ได้ ถ้าต้องการ) แล้วกดปุ่ม เริ่มสตรีมมิ่ง

รอจนกระทั่งขึ้นภาพตัวอย่างขึ้นมา แล้วกดปุ่ม Go Live! ได้เลย

เอาละ ตอนนี้ท่านก็พร้อมสำหรับการถ่ายทอดสดแล้ว เริ่มถ่ายทอดสดได้โดยการกดปุ่ม เริ่มถ่ายทอดสดได้ทันที
